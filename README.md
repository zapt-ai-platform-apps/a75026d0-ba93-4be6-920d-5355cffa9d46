# Huzaifa business

A business application built with React, Vite, and Tailwind CSS.

## Development

1. Install dependencies:
```
npm install
```

2. Start the development server:
```
npm run dev
```

3. Build for production:
```
npm run build
```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Sentry for error tracking
- Progressier for PWA functionality
